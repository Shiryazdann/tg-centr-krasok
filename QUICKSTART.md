# 🚀 БЫСТРЫЙ СТАРТ

## Шаг 1: Установка зависимостей

```bash
cd telegram-ai-bot
pip install -r requirements.txt
```

## Шаг 2: Получение токенов

### Telegram Bot Token

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя бота (например: "Центр Красок Помощник")
4. Придумайте username (например: "centr_krasok_bot")
5. Скопируйте полученный токен (формат: `123456789:ABCdefGHIjklMNOpqrs`)

### OpenAI API Key

1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в раздел **API Keys**
3. Нажмите **Create new secret key**
4. Скопируйте ключ (формат: `sk-proj-xxxxx`)
5. **Важно**: пополните баланс аккаунта ($5-10 хватит надолго)

## Шаг 3: Настройка

Создайте файл `.env` в папке проекта:

```bash
TELEGRAM_BOT_TOKEN=ваш_токен_telegram
OPENAI_API_KEY=ваш_ключ_openai
```

Пример:
```
TELEGRAM_BOT_TOKEN=7123456789:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
OPENAI_API_KEY=sk-proj-aBcDeFgHiJkLmNoPqRsTuVwXyZ123456789
```

## Шаг 4: Тестирование (опционально)

Перед запуском бота можно протестировать AI-ответы:

```bash
python test_bot.py
```

## Шаг 5: Запуск бота

```bash
python bot.py
```

Вы увидите сообщение:
```
🤖 Бот запущен и готов к работе!
Нажмите Ctrl+C для остановки
```

## Шаг 6: Тестирование в Telegram

1. Откройте Telegram
2. Найдите вашего бота по username
3. Нажмите **START**
4. Напишите вопрос, например: "Чем занимается компания?"

## 🎉 Готово!

Бот теперь отвечает на вопросы о компании.

## ⚠️ Возможные проблемы

### "ОШИБКА: Установите переменную окружения"
- Проверьте, что файл `.env` создан в папке проекта
- Проверьте правильность названий переменных

### "Unauthorized" или ошибка Telegram API
- Проверьте правильность TELEGRAM_BOT_TOKEN
- Убедитесь, что токен скопирован полностью

### "Invalid API Key" от OpenAI
- Проверьте правильность OPENAI_API_KEY
- Убедитесь, что на балансе OpenAI есть средства
- Ключ должен начинаться с `sk-`

### Бот не отвечает
- Проверьте, что скрипт bot.py запущен
- Посмотрите логи в консоли на наличие ошибок

## 📝 Дополнительно

### Изменение модели на более качественную

В файле `bot.py` строка 47:
```python
model="gpt-4o-mini"  # быстрая и дешевая
```

Замените на:
```python
model="gpt-4o"  # более умная, но дороже (~в 10 раз)
```
