## 🚀 Как запустить (кратко)

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте `.env` файл с токенами:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен
   OPENAI_API_KEY=ваш_ключ
   ```

3. Запустите бота:
   ```bash
   python bot.py
   ```

**Подробная инструкция:** см. `QUICKSTART.md`
