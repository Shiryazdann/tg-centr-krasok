# AI Telegram Assistant для "Центр Красок #1"

Telegram-бот с AI-ассистентом для ответов на вопросы о компании.

## 🎯 Возможности

- Отвечает на вопросы о компании, продукции и услугах
- Работает в формате естественного диалога (без команд)
- Использует GPT-4 для генерации ответов
- Отвечает только на основе реальной информации о компании
- Предоставляет контактную информацию и адреса

## 📋 Требования

- Python 3.8+
- Telegram Bot Token (от @BotFather)
- OpenAI API Key

## 🚀 Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Создайте Telegram бота

1. Напишите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен

### 3. Получите OpenAI API ключ

1. Зарегистрируйтесь на https://platform.openai.com
2. Создайте API ключ в разделе API Keys

### 4. Настройте переменные окружения

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Заполните файл `.env` вашими ключами:

```
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxx
```

### 5. Запустите бота

```bash
python bot.py
```

## ⚙️ Настройка

### Изменение модели GPT

В файле `bot.py` можно изменить модель:

```python
model="gpt-4o-mini"  # быстрая и дешевая
# или
model="gpt-4o"       # более качественные ответы
```

### Обновление информации о компании

Отредактируйте файл `company_data.py` для добавления новой информации.

## 💡 Как это работает

1. Пользователь отправляет сообщение в Telegram
2. Бот получает сообщение и отправляет его в OpenAI API
3. GPT анализирует вопрос и информацию о компании
4. Генерирует релевантный ответ на основе базы знаний
5. Бот отправляет ответ пользователю
